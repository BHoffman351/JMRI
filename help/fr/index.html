JMRI Système d'Aide

Ceci est la page d'aide de haut niveau JMRI.

La page de remerciements montre que beaucoup de personnes ont été en mesure de contribuer.

Utilisation du Système d'Aide

Si vous voyez ceci grâce à un outil d'aide du programme, la section à gauche vous donne trois types d'accès à l'aide de JMRI.
  • L'icône "livre" indique une Table des Matières.
  • L'icône "onglets" indique le Index.
  • La "Loupe" vous permet de rechercher le contenu
Si vous voyez ceci sur le web, les liens ci-dessus vous permettent de vous rendre àux sections correspondantes.

Information future

La page web principale de JMRI http://jmri.org est un bon point de départ pour obtenir des renseignements sur l'état actuel, les mises à jour, etc (en fonction sur votre ordinateur, vous pourriez avoir à copier ce lien dans votre navigateur Web pour être en mesure de voir des sites extérieurs correctement)

La jmri-announce liste de diffusion comporte des annonces de nouvelles versions, des changements majeurs, etc. Cette liste est destinée à avoir un très faible trafic, les débats, etc. devraient avoir lieu ailleurs. Vous pouvez vous abonner en envoyant le mot "subscribe " dans le sujet ou le corps d'un e-mail à jmri-announce-request@lists.sourceforge.net.

Nous avons un groupe Yahoo "JMRI utilisateurs" pour une discussion générale. Si vous souhaitez poser une question, obtenir des d'aide, ou de suggérer des améliorations, ce groupe de discussion est un bon endroit. Vous pouvez abonner, voir les messages, etc, à https://groups.yahoo.com/neo/groups/jmriusers.

Nospages techniques sont encore en cours d'écriture. C'est encore un stade précoce, et nous sommes à court de documentation. Mais si vous êtes intéressés en travaillant avec le code, nous serions ravis de vous aider à démarrer par e-mail, etc.
