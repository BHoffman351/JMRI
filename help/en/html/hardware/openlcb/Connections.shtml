Hardware Support: OpenLCB - Connections

This page discusses how JMRI can connect to OpenLCB.

Connecting

Via a TCH Tech CAN=USB Connector

Via a GridConnect Adapter

Via a GridConnect Network Interface

Via a Lawcell CAN USB

Via a MERG CAN-RS/-USB

You can simulate a remote OpenLCB connection, e.g. to debug things, on many computers using the "nc" command: 

nc -l 12022
