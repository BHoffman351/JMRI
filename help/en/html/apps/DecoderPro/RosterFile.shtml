Roster File Details

The Roster information presented in the selection box, and used for automatic selection of a Locomotive, is stored in the roster.xml file in the "prefs" directory. find the preferences directory using the information on the "Configuration Files" page. You can open this file and look at it to see the format. You should use the Roster menu to manipulate it to ensure that it stays self-consistent.

To move the entire roster to another computer, choose "Export Roster" from the Roster menu, select a location to save the file and then move in to the other computer before you choose "Import Roster" from the menu to import it there.

To do this manually at the file level, in the Preferences directory there will be a file called "roster.xml", and a directory called "roster". You need to move the "roster.xml" file, the "roster" directory, and all it's contents.
