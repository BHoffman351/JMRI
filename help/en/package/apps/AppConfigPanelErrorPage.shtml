JMRI: Setting The JMRI Preferences

This Help screen opens automatically when the JMRI Preferences have not been set, or there was a Preferences-related error while the program was starting up.

Once the preferences are set properly, this screen won't display. The Help pane for setting your Preferences is at main Preferences Help page. You can also open the pane by selecting "Window Help" in the Help menu of the Preferences pane. This is the help\en\package\apps\AppConfixPanelErrorPage.shtml pane
